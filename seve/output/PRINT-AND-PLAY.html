2–4 joueurs · 30–45 min · matériel : ce PDF + ~60 graines (haricots secs / perles / jetons).

1 · Le plateau — l'Anneau (12 parcelles)

Posez 4 graines dans chaque parcelle au début. Semez une parcelle non vide → déposez ses graines une par une dans les suivantes (sens horaire) → récoltez la parcelle où tombe la dernière graine.

2 · Cartes Pluie (20) — découpez

Chaque manche, révélez une carte : ajoutez une graine sur chaque parcelle marquée d'un point (9 graines). Les positions n° correspondent au plateau.

3 · Aides de jeu & feuilles de panier

Tour de jeu

  1. Manche : révélez 1 carte Pluie → 9 graines aux parcelles indiquées.
  2. Chacun, dans l'ordre : SEMER (prendre toutes les graines d'une parcelle, les déposer 1/1 horaire) puis RÉCOLTER (la dernière graine : prendre toutes les graines de sa parcelle = légumes de ce type ; elles quittent le jeu).
  3. Fin de manche : le jeton 1ᵉʳ jardinier passe à gauche.
